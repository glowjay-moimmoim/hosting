Details

클라이언트 현황

필라테스 센터

  • 운영 경력 7~8년차
  • 현재 액티브 회원 약 100명
  • 센터 확장 이력 있음
  • 기존 회원 DB 보유 (스쳐간 회원 포함)

운동복 브랜드 (카이오나)

  • 2년 전 런칭했으나 둘째 임신/출산으로 중단
  • FW 시즌 1회 진행, 이번이 두 번째 SS 시즌 준비 중
  • 이번 SS 포지션: 러닝 메인 (러닝 기반 + 필라테스/요가 겸용)
  • 촬영 5월 초 예정, 이후 예약 판매 진행 예정
  • 디자인·기획·상세페이지 모두 대표님 직접 제작

기존 마케팅 채널

  • 유튜브 (개인채널: 필라테스 희진쌤) → 운동복 노출 → 일부 구매로 이어짐
  • 스마트스토어 → 검색 유입 + 리뷰 20개↑ 확보
  • 유튜브에 쇼핑 탭 미연동 (예정)
  • 인스타 광고는 아직 없음

핵심 고민 / 페인포인트

마케팅 방향성 혼란

  • 인스타 인플루언서 방식은 따라가기 싫은데, 안 하면 안 팔릴 것 같은 딜레마
  • 남편은 '제대로 된 브랜드'를 원하고, 대표님은 유튜브 팬 기반 소수 타깃 판매를 원했으나 절충점을 못 찾은 상태

혼자서 전부 다 함

  • 기획, 디자인, 상세페이지, 운영 모두 1인 처리 → 소진 위험

예약 판매 계획 있으나 고객 확보 없음

  • 5월 초 촬영 → 예약 판매 예정이나 구매 확정된 고객 없음

마케팅 예산 400~500만 원 (제작비 제외)

  • 상세페이지 등 제작은 본인이 직접 해결

모임모임 제안 방향

제안 골자

  • 커뮤니티 브랜딩 전략 설계 + 실행 로드맵 4주 패키지
  • 금액: 150만 원 (기반이 잘 잡혀 있어서 리터치 수준으로 가능 판단)

제안 근거 — 강점

강점내용
유튜브 채널이미 운영 중, 운동복 노출로 구매 전환 경험 있음
오프라인 거점필라테스 센터 = 즉각 유입 가능한 실물 고객 존재
기존 DB과거 센터 회원 DB 보유
콘텐츠 역량디자인·기획 직접 가능
도메인 명확성임산부·필라테스·러닝 등 타깃 뚜렷함

활용 도구

  • School 플랫폼: 커뮤니티 + 구독 멤버십 + 클래스룸 통합
  • 결제: 스트라이프 (월간/연간 자동결제)
  • 레퍼럴 40% 설정 → 회원이 회원을 데려오는 구조
  • 뉴스 발행 → 전체 이메일/푸시 자동 발송
  • AI 운영체계: 브랜드 문서화 → AI 기반 콘텐츠 제작 효율화
  • 로드맵 문서화: 3개월 단위 마일스톤 설계

추천 전략 스케치

  • 유튜브 바이오에 School 커뮤니티 링크 연결 → 구독자 → 커뮤니티 유입 선순환 구조 (그린코끼리 케이스 참고)
  • 필라테스 기존 회원 → 커뮤니티 무료 초대 → 초기 커뮤니티 활성화 + 운동복 테스트 그룹 확보
  • 임산부 운동/전문가 과정 등 클래스룸 강의 상품화
  • 한강 러닝, 힐링 캠프 등 오프라인 이벤트 단발 유료 판매
  • SS 예약 판매 → 커뮤니티 내에서 선공개 + 와디즈 병행 검토
